Sorting algorithm Visualiser

python-pygame

Sorting Algorithms Sorting Algorithms are methods of reorganizing a large number of items into some specific order such as highest to lowest, or vice-versa, or even in some alphabetical order. These algorithms take an input list, processes it (i.e, performs some operations on it) and produce the sorted list

In this project ; the working of sorting algorithms like : bubble sort,insertion sort has been visualised using pygame. This project is able to show visually ; how a particular algorithm sorts.
